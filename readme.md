# PWMUSFQ

## Overview
PWMUSFQ is a mixed-signal PWM macro that integrates a 555-based timing block with a PWM generator driven by a 3-bit flash ADC. The input voltage `vin` is quantized into a 3-bit digital word and compared against a 3-bit counter, producing a PWM output whose duty cycle follows the analog input level.

## Project Description
PWMUSFQ is composed of two main sections:

### 1. 555-Based Block
The first section is based on the classical 555 structure. This block provides the timing-oriented analog functionality and preserves the spirit of the original 555 architecture while being integrated with the rest of the macro.

### 2. PWM Generator
The second section is the PWM generator. In this block, a 3-bit flash ADC receives an input voltage `vin` between 0 V and 1 V and translates it into a 3-bit digital word. That code represents the analog input level in binary form.

The 3-bit flash-ADC output is then applied to a binary comparator. The other comparator input comes from a 3-bit counter driven by `clk`. Whenever the counter value is smaller than the ADC code, the comparator output goes high; otherwise, it goes low. This directly creates a PWM waveform whose duty cycle increases as `vin` increases.

## Top-Level Pins
The VDD, VSS, and `vbias` nets are shared between the 555-based block and the flash-ADC / PWM-generator section.

| Pin Name | Type | Direction | Description |
|---|---|---|---|
| VDD | Power | Input | Shared supply voltage for the complete chip. |
| VSS | Ground | Input | Shared ground reference for the complete chip. |
| vbias | Analog bias | Input | Shared bias/reference node used by the internal analog blocks. |
| vin | Analog | Input | Analog input voltage in the 0 V to 1 V range used for duty-cycle control. |
| clk | Digital | Input | Clock input that drives the 3-bit counter. |
| PWM | Digital | Output | Top-level PWM output generated by comparing the counter value against the flash-ADC code. |
| CONTROL | Analog | Input | Control pin of the 555-based block. |

| OUTPUT | Digital | Output | Direct output node of the internal 555-based block. |

## Operating Principle
- For `vin` near 0 V, the flash ADC produces a low digital code, so the comparator is high for only a small fraction of the counter cycle. This results in a low-duty-cycle PWM output.
- For intermediate `vin` values, the ADC code increases and the PWM duty cycle increases proportionally in discrete 3-bit steps.
- For `vin` close to 1 V, the ADC output code is high, so the PWM output stays high for most of the counter period, producing a large duty cycle.
